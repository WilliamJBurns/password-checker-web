# Password Strength Checker (Web)

A web-based password strength checker with a color-coded strength bar, built with HTML, CSS, and JavaScript.

## What it does

- Checks password strength as you type
- Scores based on length, character variety, and common password detection
- Displays a color-coded bar that smoothly transitions from red (weak) to green (strong)
- Shows a score out of 7 and a tip to improve your password
- Includes a show/hide password button

## How to run it

1. Clone this repository
2. Open `index.html` in your browser

## Files

- `index.html` — the main page with HTML, CSS, and JavaScript
- `common-passwords.js` — a list of common passwords used for detection
